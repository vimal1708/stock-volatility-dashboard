# 📊 Stock Volatility Analytics Dashboard

An interactive web application to analyze financial market volatility using real-time stock data.

---

## 🚀 Features

- 📈 Stock price trend visualization
- 📉 Rolling volatility analysis (30-day)
- 📊 Multi-stock comparison (US + Indian stocks)
- 📦 Dashboard-style UI with cards
- 🔍 Automated insights & conclusions

---

## 🧰 Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Yahoo Finance API
